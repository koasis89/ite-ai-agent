# Insurance SI Proposal Templates

공제·보험 SI 제안 작업에서 반복 사용하는 템플릿 모음이다.

- `strategy-report-template.md`: RFP 분석/배점 기반 제안 전략 보고서
- `proposal-slide-outline-template.md`: 제안서 본문(PPT 장표) 스토리라인 템플릿
- `compliance-matrix-template.md`: 요구사항 수용표/기술평가항목 위치표 템플릿
- `pt-script-template.md`: 제안설명회(PT) 15~20분 발표 스크립트 템플릿

주의:
- 이 템플릿은 초안 생성용이다. 실제 제출본은 RFP 지정 서식/목차를 최우선으로 맞춘다.
- `skills/insurance-si-proposal/scripts/check_expressions.py`로 애매모호 표현을 최종 점검한다.
