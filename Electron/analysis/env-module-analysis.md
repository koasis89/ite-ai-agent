# src/main/env/ 모듈 분석

> 작성일: 2026-05-29
> 대상 경로: Electron/src/main/env/

## 1. 구성 파일

- `env-checker.ts`

## 2. 역할

`env` 모듈은 실행 전 필수 전제조건을 점검하는 안전 게이트다.
프로젝트 구조/필수 파일/런타임 전제 검증을 담당한다.

## 3. 핵심 설계 포인트

- 환경 문제를 조기 감지하여 실행 실패를 사전 차단
- 점검 결과를 IPC 또는 시스템 메시지로 전달해 사용자 가시성 확보

## 4. 검증 포인트

- 누락 파일 탐지 정확도
- 다중 OS 경로 해석 일관성
- 점검 실패 시 사용자 안내 메시지 품질
