### 2.1 스킬 패키지 이식

- `skills/insurance-si-proposal/scripts/check_expressions.py`

### 2.2 스킬 설명 보완
`skills/insurance-si-proposal/SKILL.md` 내용을 현재 프로젝트 사용 방식에 맞춰 일부 정리했습니다.

- 저장소 내 프롬프트/템플릿 경로와의 연결 정보 반영
- 프로젝트 내 활용 흐름(통합 포인트) 안내 추가

### 2.3 프롬프트 신규 추가
- `prompts/insurance-si-proposal-writer.md`

목적:
- 보험/공제 SI 제안서 작성 시 일관된 산출물을 만들기 위한 작성 지침 제공

### 2.4 템플릿 신규 추가
- `templates/insurance-si-proposal/README.md`
- `templates/insurance-si-proposal/strategy-report-template.md`

목적:

## 3. 최종 산출물 요약

### 3.1 추가된 주요 디렉터리/파일
- `skills/insurance-si-proposal/` 하위 스킬 패키지 일체
- `prompts/insurance-si-proposal-writer.md`
- `templates/insurance-si-proposal/` 하위 템플릿 일체

### 3.2 기대 효과
- 도메인 특화 스킬을 현재 저장소에서 즉시 재사용 가능
- 제안서 품질/구조 표준화
- 신규 인원 온보딩 시 문서 작성 기준점 제공

## 4. 사용 가이드(간단)

1. 제안서 초안 작성 시 `prompts/insurance-si-proposal-writer.md`를 우선 참조합니다.
2. 산출물 유형별로 `templates/insurance-si-proposal/` 템플릿을 선택해 사용합니다.
3. 표현 점검이 필요하면 `skills/insurance-si-proposal/scripts/check_expressions.py`를 활용합니다.
