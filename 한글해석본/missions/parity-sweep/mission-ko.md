# 사명
한 번의 제한된 스윕으로 `omx autoresearch`을(를) 현재 승인된 전체 패리티 계획과 패리티로 가져옵니다.

주요 대상:
- 새로운 달리기 태그가 붙은 파트
- 명시적인 `--resume <run-id>` 동작
- repo-root 활성 실행 포인터/잠금
- 신뢰할 수 있는 실행별 매니페스트 상태
- repo-root `candidate.json` 핸드오프
- 유지/폐기/모호함/오류 처리
- 재설정 안전 작업 트리-로컬 런타임 파일
- 정렬된 문서/도움말/계약/테스트

성공은 다음을 의미합니다.
1. 빌드 패스
2. 집중 패리티 테스트 통과
3. help/docs/contracts는 동일한 동작을 설명합니다.
4. 현재 PRD/테스트 사양에서 알려진 패리티 차단기가 아직 구현되지 않은 상태로 남아 있지 않습니다.
