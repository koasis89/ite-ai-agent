# 샌드박스 규칙

- 경로 검증 및 순회 거부에 초점을 맞춰 변경 사항을 유지하세요.
- 새로운 종속성이 없습니다.
- 아키텍처 변동보다 명시적인 검증 및 회귀 테스트를 선호합니다.
- 테스트를 통과하기 위해 기존 가드레일을 약화시키지 마십시오.

# 평가정책

- `pass=true`은 대상 빌드 및 MCP 경로 안전성 테스트 슬라이스가 모두 통과되었음을 의미합니다.
- `score`은 `0.00`에서 `1.00`까지 통과한 대상 검사의 비율입니다.
- 점수가 높을수록 좋습니다.
