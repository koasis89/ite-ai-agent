# 임무: MCP 경로 순회 강화

관련 없는 보안 작업으로 범위를 확장하지 않고도 MCP/상태 지향 표면 주변의 경로 검증 및 순회 방어를 강화합니다.

## 목표
안전하지 않은 경로와 순회 시도가 조치 가능한 오류와 함께 결정론적으로 거부된다는 확신이 향상됩니다.

## 중점 분야
- `src/mcp/__tests__/path-traversal.test.ts`
- `src/mcp/__tests__/state-paths.test.ts`
- 해당 테스트에서 터치된 인접한 MCP/상태 유효성 검사 코드

## 원하는 출력
다음과 같은 가장 작은 안전 경화 차이를 생성합니다.
1. 기존 허용 경로 동작을 유지합니다.
2. 필요한 경우 순회 거부를 강화합니다.
3. 대상 보안 회귀 슬라이스를 녹색으로 유지합니다.

## 성공 힌트
- 광범위한 리팩터링보다 경계 확인 및 검증 강화를 선호합니다.
- 안전하지 않은 입력을 차단하기 위해 변경이 필요하지 않은 한 현재 CLI/MCP 계약을 유지합니다.
- 작업에 의해 표면화되는 경우 의도적으로 지원되지 않는 경로 패턴을 문서화하세요.
