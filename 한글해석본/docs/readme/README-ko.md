* `../../README.md`을 표준 소스로 처리합니다.
* 저장소 루트가 아닌 이 폴더에 새 README 번역을 추가하세요.
* 표준 README와 각 지역화된 변형 간에 언어 목록을 동기화된 상태로 유지하세요.
* `docs/readme/`의 상대 링크를 유효한 상태로 유지합니다.
* 중복 파일이나 대체 명명 체계를 도입하는 대신 기존 번역을 업데이트하는 것을 선호합니다.
