# 오마이코덱스 v0.9.1

초안 작성일: 2026-03-13

빨간색 `v0.9.0` 릴리스 상태를 지우기 위해 `main`에서 빌드된 패치 핫픽스 릴리스입니다.

## 요약

`0.9.1`은 Spark Initiative 라인을 완전히 대체하는 릴리스입니다.

- `v0.9.0`은 역사적으로 빨간색으로 유지됩니다.
- `v0.9.1`은 PR [#806](https://github.com/Yeachan-Heo/oh-my-codex/pull/806)의 `dev`에 병합된 패키지 설치 연기 수화 수정 사항을 전달합니다.
- 대체 릴리스에 필요한 릴리스 핫픽스 경로 및 버전/메타데이터 업데이트 외에는 새로운 기능 범위가 도입되지 않습니다.

## 포함된 핫픽스

### 패키지 설치 연기 수화 자산이 현지화되었습니다.

패키지 설치 연기 흐름은 이제 소스 체크아웃 레이아웃에서만 유효한 경로에 의존하는 대신 수화 자산을 테스트 작업 공간으로 지역화합니다.

포함된 변경 사항:
- `scripts/smoke-packed-install.mjs`
- `scripts/__tests__/smoke-packed-install.test.mjs`

소스 기록:
- 핫픽스 커밋: `d86165d` — `fix(release): localize smoke hydration assets`
- PR [#806](https://github.com/Yeachan-Heo/oh-my-codex/pull/806)을(를) 통해 `dev`에 병합됨

## 릴리스 포지셔닝

- 기본 기능 릴리스는 **Spark Initiative**(`0.9.0`)로 유지됩니다.
