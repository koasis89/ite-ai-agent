# 릴리스 노트 — 0.12.3

## 요약

`0.12.3`은 `0.12.2`의 후속 패치입니다. `0.12.2`에 착륙할 예정이었지만 `0.12.2` 컷 후 충돌 해결을 완료한 PR [#1364](https://github.com/Yeachan-Heo/oh-my-codex/pull/1364)을 제공합니다. — `$team` 프롬프트 라우팅(Routing) 정확성 및 중복 team(멀티 worker(실제작업을 수행하는 실행단위) 인터페이스 실행 오케스트레이션) 시작 해제 — 그리고 동기화된 `0.12.3` 릴리스 자료도 포함됩니다.

## 수정 사항 및 변경 사항이 포함되었습니다.

- `UserPromptSubmit`에서 감지된 `$team`은 이제 프롬프트를 자동으로 잘못 라우팅하는 대신 루트 `team-state.json`을 시드하고 운영자를 `omx team ...` / `omx team --help`로 이동시킵니다(#1364).
- `startTeam`은 team(멀티 worker(실제작업을 수행하는 실행단위) 인터페이스 실행 오케스트레이션) 상태를 변경하거나 작업 트리를 프로비저닝하기 전에 `team_name_conflict` 오류로 중복된 활성 동일한 이름의 team(멀티 worker(실제작업을 수행하는 실행단위) 인터페이스 실행 오케스트레이션) 시작을 거부하고 기존 team(멀티 worker(실제작업을 수행하는 실행단위) 인터페이스 실행 오케스트레이션) 구성 및 작업을 그대로 유지합니다(#1364).
- 릴리스 메타데이터 및 보충 자료는 Node, Cargo, 변경 로그, 릴리스 본문 및 릴리스 준비 문서 전체에서 `0.12.3`에 맞춰 정렬됩니다.
