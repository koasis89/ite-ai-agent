# 파이프라인 템플릿

이러한 공개 템플릿은 [__TOK_0__](../github-pr-package-identity.md)에 문서화된 GitHub/PR/패키지 ID 파이프라인을 지원합니다.

- [__TOK_0__](./issue-package-identity.md): 표준 ID 블록.
- [__TOK_0__](./triage/): 일반적인 분류 결과에 대한 의견을 발행합니다.
- [__TOK_0__](./execution/): 아티팩트를 패키지, 계획, 결과, 검토 및 병합합니다.

이러한 파일에서 비밀과 내부 오케스트레이션 세부 정보를 유지하세요. 채팅은 상태를 요약할 수 있지만 GitHub 문제, PR 및 패키지 아티팩트는 정보의 소스로 남아 있습니다.
