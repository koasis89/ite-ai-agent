## 4. 신규 API 요약

```ts
// chat-viewmodel.ts
export interface ChatMessage {
  readonly id: string;
  readonly role: 'user' | 'assistant' | 'tool' | 'system';
  content: string;          // streaming 중 부분 텍스트
  readonly timestamp: number;
  streaming: boolean;
}
class ChatViewModel {
  constructor(opts?: { ringLimit?: number; idFactory?: () => string; now?: () => number });
  list(): readonly ChatMessage[];
  size(): number;
  append(role, content): ChatMessage;
  streamStart(role?): ChatMessage;       // 빈 본문, streaming=true
  streamAppend(id, token): boolean;       // 누적
  streamEnd(id): boolean;                 // streaming=false
  clear(): void;
  onChange(handler): Unsubscribe;
}

// markdown.ts
export function renderMarkdownSafe(input: string): string; // escape 된 HTML

// chat-panel.ts
class ChatPanel {
  constructor(elements: { messageList; inputBox; sendButton; clearButton? },
              options?: { viewModel?; onSlashCommand?; onUserMessage? });
  viewModel(): ChatViewModel;
  dispose(): void;
}
```

### 보안 정책 (실제 구현)
- 마크다운 렌더는 **모든 본문을 escape 후 화이트리스트 토큰만 재구성** — raw HTML 문자열을 신뢰하지 않음.
- 지원 토큰: fenced code(```lang), inline code, 헤딩(#~######), ul/ol, **strong**, *em* / _em_, 링크.
- 링크는 `http(s)`/`mailto`/`#frag` 만 허용 — `javascript:`/`data:`/`vbscript:`/`file:` 거부 → 원본을 텍스트로 escape.
- 외부 링크는 `target="_blank" rel="noopener noreferrer"` 강제.
- 이미지/스크립트/iframe 등 모든 HTML 태그는 화이트리스트 외이므로 escape 되어 텍스트로만 노출.
