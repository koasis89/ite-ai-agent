## 2. 목표

데스크탑 명령창에서 `omx` CLI 서브커맨드 3종(`doctor`, `adapt openclaw probe --json`, `state list-active --json`)을
**고정된 보안 정책** 아래에서 외부 프로세스로 트리거하고, 결과(`exitCode`, `stdout`, `durationMs`, 절단 플래그)를
IPC 응답으로 반환한다.

## 3. 변경 내역

### 3.1 코드 (`desktop/ipc/commands.ts`)
- `allowedCommands` 3종 추가 — `omx_doctor`, `omx_adapt_probe`, `omx_state_status` (총 15종).
- `omxCliMatrix` 매핑 테이블 — 사용자 args 를 무시하고 코드 상수 인자 배열을 사용.
  - `omx_doctor` → `[dist/cli/omx.js, doctor]`
  - `omx_adapt_probe` → `[dist/cli/omx.js, adapt, openclaw, probe, --json]`
  - `omx_state_status` → `[dist/cli/omx.js, state, list-active, --json]`
- `runOmxSubcommand(args)` helper — `child_process.spawn(process.execPath, args, { cwd: process.cwd(), shell:false, windowsHide:true })`.
- 30초 watchdog (`setTimeout` → `SIGKILL`), stdout/stderr 8KB 절단 + 절단 플래그.
- `isOmxExecAllowed()` — 환경변수 `OMX_DESKTOP_ALLOW_EXEC=0|false` 시 차단.
- switch case 3종 추가 (단일 분기에 fall-through 통합).
