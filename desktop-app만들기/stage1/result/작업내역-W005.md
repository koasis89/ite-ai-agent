## 4. 실제 변경 내역

### 4.1 Electron 실행/빌드 경로 추가
`package.json`에 아래 스크립트를 추가했다.
- `build:desktop`: `tsconfig.desktop.json` 기준 TypeScript 컴파일 + 렌더러 HTML 복사
- `desktop:dev`: `build:desktop` 후 `electron dist-desktop/main/index.js` 실행

또한 `devDependencies`에 `electron`을 추가했다.

### 4.2 Desktop 엔트리 파일 스캐폴딩
1) `desktop/main/index.ts`
- BrowserWindow 1개 생성
- `ready-to-show` 이후 창 표시
- 보안 옵션 적용
  - `contextIsolation: true`
  - `nodeIntegration: false`
  - `sandbox: true`
- preload 경로를 `../preload/index.js`로 고정

2) `desktop/preload/index.ts`
- `contextBridge.exposeInMainWorld`로 안전한 읽기 전용 상태 노출
- `window.omxDesktop`에 preload 로드 여부와 버전 정보 제공

3) `desktop/renderer/index.html`, `desktop/renderer/app.ts`
- 최소 화면 + 상태 출력 영역 구성
- `app.ts`에서 `window.omxDesktop` 값을 JSON으로 렌더하여 preload 로드 상태를 표시

4) `desktop/scripts/copy-assets.mjs`
- `desktop/renderer/index.html`을 `dist-desktop/renderer/index.html`로 복사
- `build:desktop`에서 호출되도록 구성

5) `tsconfig.desktop.json`
- `desktop/**/*.ts` 전용 컴파일 구성 추가
- 출력 디렉터리: `dist-desktop`

### 4.3 기준 문서 동기화
